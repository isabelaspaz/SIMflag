# SIMflag

Projeto desenvolvido com Expo/React Native.

## Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/isabelaspaz/SIMflag.git
```

### 2. Acessar a pasta do projeto

```bash
cd SIMflag
```

### 3. Instalar dependências:

```bash
npm install
```

### 4. Iniciar o projeto:

```bash
npx expo start
```

## Observações:
- É necessário ter o Node.js instalado na máquina.
- Também é recomendado ter o Expo Go no celular ou um emulador configurado, caso queira testar no dispositivo móvel.
- Após iniciar, o Expo abrirá opções para executar no navegador, em emulador ou no celular via QR Code.
